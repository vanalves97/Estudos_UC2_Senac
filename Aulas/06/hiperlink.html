Introdução aos Hiperlinks em HTML

Hiperlinks são um dos elementos fundamentais da web, permitindo a navegação entre páginas e recursos. Em HTML, os hiperlinks são criados usando a tag "a", também conhecida como "anchor" (âncora). Essa tag pode vincular o usuario a outra paginas, arquivos, ou até mesmo seçoes especificas dentro da mesma página.

A tag "a"

A tag "a" é usada para criar hiperlinks. Ela pode incluir varios atributos mas o mais comuns são href e traget.

Sintaxe Básica da Tag "a"

Texto do Link

Atributos Comuns da Tag "a":

  • href: Define o destino do link (URL)
  • traget: Especifica onde abrir o link. Valores comuns são _self (padrão) e _blank (nova aba/janela)
  • Title: Adicona um texto Explicativo que aparece ao passar o mouse sobre o link.

Navegação Dentro da Mesma pagina

Para criar links que naveguem para diferentes seções da mesma pagina, usamos identificadores (id) e o atributo href com #.
