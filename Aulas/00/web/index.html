HTML

HTML (linguagem de Marcação de Hipertexto) é o bloco de construção mais basico da web. Define o significado e a assinatura de conteudo web.

TAG ou Margação

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador web.

Um elemento HTML é separado de outro texto em um documento por "tags"" As tag consistem no nome do elemneto entre um "<" e ">"
